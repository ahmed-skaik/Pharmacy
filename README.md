## 📖 Project Overview

The Al-Azhar Pharmacy Portal was developed to address a real operational problem: academic materials were scattered across multiple platforms, making access inefficient for students.

This platform centralizes:

- E-learning lectures
- Course summaries
- Previous exams
- Faculty-related academic information
- Structured navigation per academic year

The website was publicly launched in 2022 and actively used by students within the faculty.

## 🏗 Architecture & Technical Approach

This version (V1) is a fully static frontend application built with a focus on clarity, performance, and maintainability.

### Core Stack

- **HTML5** – Semantic structure and accessibility
- **CSS3** – Custom styling
- **SASS** – Modular and maintainable styling architecture
- **Bootstrap** – Responsive layout system
- **JavaScript (ES6+)** – Interactive UI behavior
- **Font Awesome** – Icon system integration

The project emphasizes structured layout organization and reusable styling components using SASS.

---

## 🚀 Deployment

- Source code hosted on GitHub
- Deployed using **GitHub Pages**
- Connected to a custom domain (alazharpharmacy.com)
- Domain originally registered via Google Domains (now Squarespace Domains)

The deployment pipeline reflects a simple but reliable static hosting strategy suitable for academic environments.

### 📊 Analytics & Monitoring

Google Analytics was integrated to monitor real-world usage patterns, including:

- Traffic volume
- Most visited pages
- User behavior insights
- Geographic distribution

This data helped evaluate platform reach and understand how students interacted with academic content.

---

## 📱 Responsive & Accessibility-Focused Design

The platform was designed with mobile-first considerations, recognizing that many students rely primarily on smartphones.

Key priorities:
- Clear navigation hierarchy
- Readable typography
- Lightweight performance
- Minimal cognitive friction
